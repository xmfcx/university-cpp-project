# university-cpp-project

## Build
### Requirements
* gnu c++ compiler
* cmake 3.5 (minimum)
### Build Instructions
To build the application, follow these steps:
cd .../university-cpp-project/
mkdir build
cd build
cmake ..
make -j4
That's it, you now should have the university executable file in your build directory.
